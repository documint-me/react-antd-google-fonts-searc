# React Ant Design Google Fonts Search 

> Search and select google fonts and get output in various configurations.

## Prerequisites

This project requires NodeJS (version 8 or later) and NPM.
[Node](http://nodejs.org/) and [NPM](https://npmjs.org/) are really easy to install.
To make sure you have them available on your machine,
try running the following command.

```sh
$ npm -v && node -v
6.4.1
v8.16.0
```

This project has been created using Ant Design, Ant Design Icons and Lodash and will therefore require these packages when installing.

## Installation

**BEFORE YOU INSTALL:** please read the [prerequisites](#prerequisites)

Start with cloning this repo on your local machine:

```sh
$ git clone https://github.com/ORG/PROJECT.git
$ cd PROJECT
```

To install and set up the library, run:

```sh
$ npm install @documint/react-antd-google-fonts-search
```

Or if you prefer using Yarn:

```sh
$ yarn add @documint/react-antd-google-fonts-search
```
